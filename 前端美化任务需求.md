## 1. 美学方向

**选定方案**: D — 水墨留白 · 东方极简

### 设计理念

灵感来自中国传统水墨画的留白美学，克制、雅致、富有艺术气息。与"汉唐剧院"品牌名称高度契合，在票务类产品中独树一帜。

### 配色系统

| Token | 色值 | 用途 |
|-------|------|------|
| 墨黑 | `#2c2c2c` | 正文、标题 |
| 石灰 | `#666666` | 次级文字 |
| 浅墨 | `#999999` | 辅助文字、禁用态 |
| 宣纸白 | `#FAFAF7` | 页面背景（暖白带微微黄调） |
| 米灰 | `#F5F0EB` | 卡片背景、区域底色 |
| 暖灰 | `#E8E0D5` | 分割线、边框 |
| 点缀色 | `#C9A96E` | 暗金 — 仅用于高亮状态、选中态、CTA 按钮 |

### 排版

- **标题**: 衬线字体（Noto Serif SC / Georgia），字间距宽松
- **正文**: 无衬线字体（系统默认），字号 14-15px
- **间距**: 大留白，内容区不拥挤，卡片间距 16-24px

### 造型

- **圆角**: 直角或微圆角 2-4px（非大圆角）
- **阴影**: 极少使用，依赖边框和留白区分层级
- **按钮**: 线性边框风格，hover 时填充墨色
- **动画**: 克制 — 仅 hover 过渡和页面切换淡入

## 2. 技术方案

### 新增依赖

- `tailwindcss` + `@tailwindcss/vite` — Tailwind CSS v4 with Vite plugin
- 字体: `Noto Serif SC` (Google Fonts / 自托管) 用于标题

### 样式架构

```
全局 CSS (globals.css)
  ├── @tailwind 指令
  ├── 自定义主题 CSS 变量
  └── 基础样式重置/字体

Tailwind Config (tailwind.config.ts)
  ├── colors → 墨色系 + 宣纸白 + 暗金点缀
  ├── fontFamily → serif (标题) + sans (正文)
  └── extend → 自定义间距、动画

Ant Design ConfigProvider theme
  └── token 与 Tailwind 设计系统保持一致
```

### 样式策略

- **布局/间距/颜色**: 优先使用 Tailwind 原子类
- **Ant Design 组件**: 通过 `className` 注入 Tailwind 类，必要时用 `style` 覆盖
- **自定义组件**: 纯 Tailwind 构建

## 3. 任务拆分

### 阶段一：基础设施

#### T01 — 安装 Tailwind CSS

- 安装 `tailwindcss @tailwindcss/vite`
- 配置 Vite 插件
- 验证 Tailwind 编译正常

#### T02 — 配置自定义主题

- 创建 `tailwind.config.ts`
- 定义 colors: `ink`, `stone`, `rice`, `warm`, `gold` 色阶
- 定义 fontFamily: `serif` → Noto Serif SC, `sans` → system
- 扩展 spacing、borderRadius

#### T03 — 创建全局 CSS

- 创建 `src/styles/globals.css`
- `@import "tailwindcss"` 指令
- `@theme` 块定义自定义 tokens
- 基础样式重置
- 在 `main.tsx` 中引入

#### T04 — ConfigProvider 主题同步

- 修改 `App.tsx` 中 antd ConfigProvider 的 theme token
- 主色、文字色、背景色、圆角与 Tailwind 系统一致

### 阶段二：布局壳

#### T05 — Header 导航栏

- 文件: `src/layouts/CustomerLayout.tsx`
- 宣纸白背景、底部 1px 暖灰细线
- 品牌名用衬线字体、宽字间距
- 导航链接墨灰色，当前页底部 2px 下划线
- 登录/注册按钮线性边框风格

#### T06 — Footer 页脚

- 同文件 `CustomerLayout.tsx`
- 顶部 1px 暖灰细线
- 简洁居中文字：版权信息、浅墨色

#### T07 — 全局背景 + 内容区

- 页面底色 `bg-[#FAFAF7]`
- 内容区 `max-w-6xl mx-auto` 居中
- 足够的 padding 留白

### 阶段三：购票核心链路

#### T08 — 首页 Home

- 文件: `src/pages/customer/Home.tsx`
- Hero 区：大留白 + 衬线标题 + 暗金点缀 CTA
- 热卖剧目：2-3列卡片网格，卡片边框细线、海报图占视觉焦点
- 近期演出：简洁列表，日期+剧目名+购票按钮

#### T09 — 排片 Schedule

- 文件: `src/pages/customer/Schedule.tsx`
- 筛选栏：去掉复杂 Select 框，改为简洁标签/按钮组
- 日期选择器风格统一为线性风格
- 演出卡片列表：细边框卡片，左海报右信息

#### T10 — 选座 Seats

- 文件: `src/pages/customer/Seats.tsx`
- 座位图：可用=暖灰边框空心、已选=墨色填充、已售=浅墨叉号
- 舞台区域：衬线字体标识
- 倒计时：暗金色强调
- 底部信息栏：选中座位 + 总价 + 确认按钮

#### T11 — 订单确认 Order

- 文件: `src/pages/customer/Order.tsx`
