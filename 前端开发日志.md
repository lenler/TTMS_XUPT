# TTMS 前端开发日志

## 2026-05-25：项目脚手架搭建（feat/frontend-init）

### 完成内容

| 项目 | 详情 |
|------|------|
| 分支 | `feat/frontend-init`（从 `main` 创建） |
| 脚手架 | `pnpm create vite ttms-frontend --template react-ts` |
| 依赖安装 | react-router-dom, antd, @ant-design/icons, axios, zustand, msw |
| Vite 配置 | `@` 路径别名 → `src/`，开发服务器端口 3000 |
| TypeScript 配置 | 严格模式，路径别名 `@/*` → `src/*`，关闭 erasableSyntaxOnly（支持 enum） |
| MSW 初始化 | `npx msw init public/`，生成 Service Worker |
| 环境变量 | `.env` 中设置 `VITE_ENABLE_MOCK=true` |

### 目录结构

完整创建了 73 个源文件，覆盖：

- **types/** — `api.ts`（通用响应/分页类型）、`models.ts`（7 个业务实体类型）、`enums.ts`（3 个状态枚举）
- **services/request.ts** — Axios 实例，请求/响应拦截器，token 注入
- **services/admin/** — 10 个 API 服务文件（auth/studio/play/schedule/check/sale/employee/customer/role/finance）
- **services/customer/** — 5 个 API 服务文件（auth/home/schedule/order/board）
- **stores/** — `authStore.ts`（token/用户/菜单持久化）、`cartStore.ts`（选座购物车）
- **routes/** — `admin.routes.tsx`、`customer.routes.tsx`
- **layouts/** — `AdminLayout.tsx`（SaaS 侧边菜单布局）、`CustomerLayout.tsx`（顶栏导航布局）
- **pages/admin/** — 登录页、工作台、11 个模块列表/详情占位页
- **pages/customer/** — 首页、放映安排、选座、订单、支付结果、我的订单、榜单、联系我们、登录、注册
- **components/** — PageTable、SearchForm、StatusTag、SeatEditor 占位
- **hooks/** — useCRUD、usePagination、useAuth 占位
- **mocks/handlers/** — 11 个模块 Handler 占位文件
- **mocks/data/** — 7 个 JSON 数据占位文件
