# TTMS_XUPT 项目总览

本仓库按运行工程、课程资料和归档文件分块管理。

## 目录说明

| 路径 | 内容 |
|------|------|
| `ttms/` | Spring Boot 后端、数据库脚本、项目内部文档 |
| `ttms-frontend/` | 当前 React + Vite 前端页面 |
| `project-docs/reference/` | 需求分析、软件设计、接口设计等原始课程资料 |
| `project-docs/logs/` | 前端、后端、文档开发日志 |
| `project-docs/guides/` | 开发前必读、协作说明 |
| `archives/` | 本地压缩包等归档文件，不提交 Git |

## 快速入口

- 后端说明：`ttms/README.md`
- 架构文档：`ttms/docs/architecture.md`
- 接口文档：`ttms/docs/api.md`
- 成员 B 后端开发文档：`ttms/docs/member-b-development.md`
- 核心需求与设计资料：`project-docs/reference/TTMS软件设计与需求分析.md`
